# Portfolio v4

My current personal portfolio website showcasing all my projects and experience.

## Tech Stack

- **Astro**
- **Preact**
- **Tailwind CSS**
- **TypeScript**

## Development

```bash
npm install
npm run dev     # Start dev server at localhost:4321
npm run build   # Production build
npm run preview # Preview production build
```

## Styling Conventions

Use semantic color tokens defined in `global.css`:
- `--color-accent` for highlights
- `--color-background` for page background
- `--color-text` for primary text
- `--color-muted` for secondary text
